Cara Mendapatkan URL Avatar

  1. Buka situs Qu.ax atau Imgur.

  3. Setelah selesai diupload, copy URL gambar (Pastikan link berakhir dengan `.jpg`, `.png`, dll.).
  4. Tempel (Paste) URL tersebut pada kolom Avatar URL Publik di atas.
